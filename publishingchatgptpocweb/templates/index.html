How does it work?

  • Data Processing

    Raw data is cleaned, processed and converted into small and summarised documents

  • Data Vectorization

    Processed documents are then converted into vectors embeddings using an AI embedding tool. This is basically converting your textual data into numeric patterns easy to understand for a large language model (LLM) like ChatGPT

  • Search, Read and Enjoy

    Your questions are then passed to an LLM which finds the most relevant documents with vector pattern matching and forms up the final response
